# 风格锚点拆解范例

## 什么是风格锚点

将模糊的「风格描述」拆为可被 AI 精确执行的**多维指令**。
每一个锚点 = 一个维度 + 一句具体指令。
多个锚点锁定在一起 = 精确的风格复现。

## 视觉类：7 维锚点

```
视角 → 前景 → 主体 → 构图/排版 → 元素风格 → 色调 → 氛围
```

### 范例 A：暖调鱼眼产品冲击

```
视角：超广角鱼眼镜头，畸变集中在画面中央产品区，边缘拉伸形成张力
前景：产品关联碎屑散落，形成浅景深模糊前景层
主体：主体正面直视镜头，与产品有肢体互动
构图：主标题置于上方 1/3 处，半透明底条；副标题置于下方
元素风格：标题使用粗圆体/胖体 + 文字描边；副标题细体灰色
色调：暖黄基调，色温约 4500K，微橙色偏色，产品区局部提亮
氛围：高能量、亲近感，仿佛第一人称视角
```

### 范例 B：极简白底产品

```
视角：标准产品摄影视角，45° 俯拍，镜头高度与产品中线平齐
前景：无，产品主体完全孤立
主体：产品居中，材质高光/折射清晰可见
构图：产品居中，文字信息置于画面外或下方纯色区
元素风格：无衬线轻体字，字间距宽松
色调：纯白背景 #FFFFFF，产品自带色为唯一色彩来源
氛围：干净、克制、无干扰
```

### 范例 C：日系胶片户外

```
视角：略微仰拍，模拟手持手机的自然视角
前景：场景边缘入画形成自然画框（桌面/树叶/窗框）
主体：手部入镜与产品互动，做出递向镜头的动作
构图：竖排大字居中，描边字确保复杂背景可读
元素风格：手写体/毛笔字风格
色调：日系胶片，略微过曝 +1/3 档，高光偏青阴影偏暖
氛围：松弛、日常、不经意
```

### 范例 D：国潮插画 + 实物

```
视角：正面平视，产品平铺展示
前景：插画风格的传统纹样环绕产品
主体：产品细节清晰，纹理可见
构图：右上角竖排标题，传统排版方向
元素风格：衬线书法字体 + 印章风格角标
色调：中国红 + 藏蓝 + 烫金，背景米白宣纸质感
氛围：文化底蕴、仪式感
```

## 文字类：6 维锚点

```
语调 → 句式 → 用词 → 节奏 → 人称 → 信息密度
```

### 范例 E：科技深度分析

```
语调：冷静客观，不下结论但给出倾向
句式：中长句为主，短句做转折钩子，避免排比句
用词：精确术语 + 大白话解释，不用「震撼」「颠覆」
节奏：三段式——场景切入 → 技术拆解 → 趋势判断
人称：第一人称复数（「我们来看看」），非「小编」
信息密度：每段至少一个数据点或具体案例，不空泛
```

### 范例 F：生活种草

```
语调：闺蜜聊天感，有情绪起伏但不过度亢奋
句式：短句为主，一句一段，多用疑问句钩子
用词：口语化 + emoji + 网络用语，不用书面语
节奏：痛点 → 发现 → 体验 → 对比 → 推荐
人称：第一人称单数（「我用了三周发现」）
信息密度：细节密集（质地/气味/使用感），不写笼统好评
```

## 锚点编写通用检查清单

适用于任何维度，任何内容类型：

- [ ] 每个维度 ≤ 30 字，不展开
- [ ] 零抽象形容词——用可测量/可观察的描述替代
- [ ] 维度之间不矛盾
- [ ] 用目标 AI 模型的母语词汇（图像模型用摄影/绘画词；文本模型用文学/语言学词）
- [ ] 至少覆盖 5 个维度（越多越锁定）

## 抽象词 → 具体指令 对照表

| 抽象词 | 转化为具体指令 |
|--------|---------------|
| 高级感 | 单色或低饱和色调 / 大面积留白 / 单一光源 / 材质高光可见 |
| 冲击力 | 超广角 / 高对比度 / 主体占画面 70%+ / 动感模糊边缘 |
| 日系 | 色温 5000K+ / 高光偏青 / 略微过曝 / 浅景深 |
| 复古 | 色温 3200K-4000K / 暗角 / 胶片颗粒 / 高光偏黄 |
| 科技感 | 冷色调主导 / 金属材质 / 边缘光 / 无衬线几何字体 |
| 温暖 | 色温 3000K-4500K / 侧逆光 / 柔焦 / 暖色占比 > 60% |
| 干净 | 单色背景 / 无多余元素 / 产品占比 > 50% / 光影柔和 |
| 接地气 | 正面平视 / 自然光 / 非专业模特感 / 环境有生活痕迹 |
