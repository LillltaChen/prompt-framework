# 通用提示词框架 · 架构总览

> 不绑定 平台 · 品类 · 风格 · 模型 —— 元框架

---

## 核心机制

```
骨架（一次定义） + 变量（每次填充） = 最终提示词（代入输出）
     ↖ 结构·可复用          ↗ 内容·可追溯
```

---

## 五阶段流水线

### 阶段 0 · 确定骨架

| 模式 A · 预设骨架 | 模式 B · 自定义骨架 |
|---|---|
| SK-1 视觉画面生成 | L1 角色与任务层 `{ROLE} {TASK} {FORMAT}` |
| SK-2 电商产品封面 | L2 风格与约束层 `{STYLE} {MUST_INCLUDE} {MUST_AVOID}` |
| SK-3 长文写作 | L3 内容与细节层 `{SUBJECT} {ACTION} {SCENE} {PROP} {TEXT} {ACCENT}` |
| SK-4 视频脚本 | |

### 阶段 1 · 变量填充

| 核心变量 · 用户确认 | 衍生变量 · Claude 推导 |
|---|---|
| 内容主体 | 尺寸 / 比例 |
| 核心风格 / 调性 | 构图 / 视角 |
| 关键约束 | 色彩 / 色调 |
| 文字 / 文案 | 边缘细节 |

> 填充规则：中文 · 具体可视化 · 单维度 · 无行业黑话 · 不适用则填「无」

### 阶段 2 · 风格锚点拆解

| 类型 | 维度管道 |
|---|---|
| **视觉 7 维** | 视角 → 前景 → 主体 → 构图 → 元素风格 → 色调 → 氛围 |
| **文字 6 维** | 语调 → 句式 → 用词 → 节奏 → 人称 → 信息密度 |

> 三条铁律：≤ 30 字/条 · 零抽象形容词 · AI 可直接执行

### 阶段 3 · 输出

| 输出项 | 说明 |
|---|---|
| 🎯 **最终提示词** | 代入后完整版 · 中英双语 |
| 📋 **变量回执** | 全部变量最终值 · 可追溯至源头 |
| 🔧 **使用建议** | 推荐模型 + 参数 + 补充说明 |

### 阶段 4 · 自检清单

- [ ] 全部变量已填充 · 无 `{ }` 占位符残留
- [ ] 核心变量均来自用户输入 · 未擅自替换
- [ ] 风格锚点覆盖 ≥ 5 个维度
- [ ] 每条锚点 ≤ 30 字 · 无抽象形容词
- [ ] 内容可被目标模型直接执行
- [ ] 正负向约束不矛盾

## 设计原则

| # | 原则 | 说明 |
|---|---|---|
| ① | **骨架变量分离** | 骨架定义结构（一次），变量提供内容（每次），换变量不换骨架 |
| ② | **变量可追溯** | 每个最终值可回溯到用户输入或推导逻辑 |
| ③ | **风格可执行** | 不依赖「高级」「好看」等主观词，AI 可直接执行 |
| ④ | **框架不绑定内容** | 图像生成 / 文章写作 / 视频脚本均适用同一套机制 |

## 抽象词 → 具体指令

| 抽象词 | 具体指令 |
|---|---|
| 高级感 | 单色 / 大面积留白 / 单一光源 / 材质高光可见 |
| 冲击力 | 超广角 / 高对比度 / 主体占 70%+ / 动感模糊 |
| 日系 | 色温 5000K+ / 高光偏青 / 略微过曝 / 浅景深 |
| 复古 | 色温 3200-4000K / 暗角 / 胶片颗粒 / 高光偏黄 |
| 科技感 | 冷色调 / 金属材质 / 边缘光 / 无衬线几何字体 |
| 温暖 | 色温 3000-4500K / 侧逆光 / 柔焦 / 暖色占比 60%+ |
| 干净 | 单色背景 / 无多余元素 / 产品占比 50%+ / 光影柔和 |
| 接地气 | 正面平视 / 自然光 / 非模特感 / 生活痕迹 |
