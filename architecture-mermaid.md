# 通用提示词框架 · 架构总览

> 不绑定 平台 · 品类 · 风格 · 模型 —— 元框架

## 核心公式

```mermaid
graph LR
    A[骨架<br/>结构 · 一次定义] -->|+| B[变量<br/>内容 · 每次填充]
    B -->|" ="| C[最终提示词<br/>代入后完整输出]
```

## 五阶段流水线

```mermaid
graph TB
    P0[阶段 0<br/>确定骨架] --> P1[阶段 1<br/>变量填充]
    P1 --> P2[阶段 2<br/>风格锚点拆解]
    P2 --> P3[阶段 3<br/>输出]
    P3 --> P4[阶段 4<br/>自检]
```

### 阶段 0 · 确定骨架

```mermaid
graph TB
    A[模式 A<br/>预设骨架] --> A1[SK-1 视觉画面生成]
    A --> A2[SK-2 电商产品封面]
    A --> A3[SK-3 长文写作]
    A --> A4[SK-4 视频脚本]

    B[模式 B<br/>自定义骨架] --> B1[L1 角色与任务层]
    B --> B2[L2 风格与约束层]
    B --> B3[L3 内容与细节层]
```

### 阶段 1 · 变量填充

```mermaid
graph LR
    subgraph 核心变量 · 用户确认
        C1[内容主体]
        C2[风格调性]
        C3[关键约束]
        C4[文字文案]
    end
    subgraph 衍生变量 · Claude 推导
        D1[尺寸比例]
        D2[构图视角]
        D3[色彩色调]
        D4[边缘细节]
    end
```

### 阶段 2 · 风格锚点拆解

```mermaid
graph LR
    subgraph 视觉 7 维
        V1[视角] --> V2[前景] --> V3[主体] --> V4[构图] --> V5[元素风格] --> V6[色调] --> V7[氛围]
    end
```

```mermaid
graph LR
    subgraph 文字 6 维
        T1[语调] --> T2[句式] --> T3[用词] --> T4[节奏] --> T5[人称] --> T6[信息密度]
    end
```

> 三条铁律：≤ 30 字/条 · 零抽象形容词 · AI 可直接执行

### 阶段 3 · 输出

```mermaid
graph TB
    O1[🎯 最终提示词<br/>中英双语]
    O2[📋 变量回执<br/>可追溯至源头]
    O3[🔧 使用建议<br/>推荐模型 + 参数]
```

### 阶段 4 · 自检

```mermaid
graph TB
    S1[变量全填充 · 无残留]
    S2[核心变量未替换]
    S3[锚点 ≥ 5 维]
    S4[≤ 30 字 · 无抽象词]
    S5[AI 可直接执行]
    S6[正负约束不矛盾]
```

## 设计原则

```mermaid
graph TB
    PR1[① 骨架变量分离<br/>结构一次定义，内容每次填充]
    PR2[② 变量可追溯<br/>每个值可回溯到用户输入]
    PR3[③ 风格可执行<br/>不依赖主观词，AI 直接执行]
    PR4[④ 框架不绑定内容<br/>图像/文章/视频脚本均适用]
```

## 抽象词 → 具体指令

| 抽象词 | 具体指令 |
|---|---|
| 高级感 | 单色 / 大面积留白 / 单一光源 / 材质高光可见 |
| 冲击力 | 超广角 / 高对比度 / 主体占 70%+ / 动感模糊 |
| 日系 | 色温 5000K+ / 高光偏青 / 略微过曝 / 浅景深 |
| 复古 | 暗角 / 胶片颗粒 / 高光偏黄 |
| 科技感 | 冷色调 / 金属材质 / 边缘光 / 无衬线几何字体 |
| 温暖 | 色温 3000-4500K / 侧逆光 / 柔焦 |
| 干净 | 单色背景 / 无多余元素 / 光影柔和 |
| 接地气 | 正面平视 / 自然光 / 生活痕迹 |
