# 快速使用指南

一分钟上手，三分钟出图。

---

## 这句话就能触发

对 Claude 说：

> 帮我生成一张宠物猫粮的小红书封面提示词

或者：

> 我想写一篇 AI 工具评测的公众号文章，帮我结构化出 prompt

---

## 三种用法

### 用法 A：一句话丢需求

直接把你的创意需求说出来，Claude 走完 5 阶段流水线，输出完整提示词。

```
你：「帮我生成一张精华液的抖音种草封面提示词，要极简白底风格」
   → 自动匹配骨架 2，填变量，拆锚点，输出中英双语 prompt
```

### 用法 B：指定骨架

如果你知道要什么类型的输出，告诉 Claude 用哪个骨架。

```
你：「用骨架 3（长文写作），帮我写一篇 RAG 技术科普文章的大纲 prompt」
   → 直接用骨架 3 填变量，不走骨架选择这一步
```

**4 个预设骨架：**

| 编号 | 骨架 | 一句话 |
|------|------|--------|
| SK-1 | 视觉画面生成 | 海报 / 插画 / 摄影 |
| SK-2 | 电商产品封面 | 带货 / 种草主图 |
| SK-3 | 长文写作 | 公众号 / 博客 / 教程 |
| SK-4 | 视频脚本 | 短视频 / 口播 / 产品演示 |

### 用法 C：自定义骨架

需求特殊，骨架库里没有？告诉 Claude 你的输出需要什么层次。

```
你：「我要一个播客节目介绍的 prompt，需要包含：主播人设、节目定位、单集结构、开头钩子」
   → Claude 现场搭三层骨架，填变量，输出
```

## 实际例子

### 输入

> 帮我想一个咖啡液产品的抖音短视频封面 prompt，要日系户外风格，主文案「3秒速溶」，副文案「冷热双泡」

### Claude 走完流程后输出

**🎯 最终提示词**

```
# 角色
你是一位「日系胶片户外产品风格」的锁定执行专家，专精抖音电商食品饮料封面生成。

# 任务
创作一张 9:16 比例的抖音食品饮料种草封面，采用「日系胶片户外产品风格」视觉系统。

# 风格保真锁定
视角：略微仰拍，模拟手持手机的自然视角
前景：木质野餐桌边缘入画形成自然画框
主体：一只手端着咖啡杯入镜，杯中为深色咖啡液
排版：竖排大字居中，描边字确保背景上可读
元素风格：手写体标题 + 圆角标签
色调：日系胶片，色温 5000K+，略微过曝，高光偏青
氛围：松弛户外感，晨光柔和的日常场景

# 画面内容
主体：一只手自然端着透明咖啡杯，咖啡液正在溶解扩散
动作：手部递向镜头，杯中液体形成溶解漩涡
产品：XX 品牌咖啡液条状包装，撕开一角置于杯旁
场景：户外木质野餐桌上，晨光斜照
背景：虚化的绿植和天空，形成浅景深
主文案：3秒速溶
副文案：冷热双泡
点缀：杯口轻烟/水珠，角落圆形「懒人必备」标签

# 必须保留
咖啡液条状包装的品牌色和 logo

# 严格避免
无

# 合规声明
不涉及功效宣称；不含绝对化用语；画面为创意表达非实际产品效果
```

**📋 变量回执**

| # | 变量 | 类型 | 值 |
|---|------|------|-----|
| 1 | STYLE_NAME | 核心 | 日系胶片户外产品风格 |
| 2 | PLATFORM | 核心 | 抖音 |
| 3 | CONTENT_TYPE | 核心 | 食品饮料 |
| 4 | SUBJECT | 核心 | 一只手端着透明咖啡杯 |
| 5 | PRODUCT_OR_PROP | 核心 | XX 品牌咖啡液条状包装 |
| 6 | MAIN_TEXT | 核心 | 3秒速溶 |
| 7 | SECONDARY_TEXT | 核心 | 冷热双泡 |
| 8 | ASPECT_RATIO | 衍生 | 9:16 |
| 9 | STYLE_FIDELITY_ANCHORS | 衍生 | 视角/前景/主体/排版/元素/色调/氛围 |
| 10 | SUBJECT_ACTION | 衍生 | 手递向镜头，液体溶解漩涡 |
| ... | ... | ... | ... |

**🔧 使用建议**
- 推荐模型：Flux / Midjourney / 即梦
- 建议尺寸：1080×1920px
- 补充说明：咖啡液的实际包装色和 logo 需替换为真实产品素材

## 和直接写 prompt 的区别

| 直接写 prompt | 用这个框架 |
|---|---|
| 每次从零想结构 | 骨架复用，只改变量 |
| 漏掉关键维度 | 自检清单兜底 |
| 风格描述模糊（日系/高级感） | 锚点拆成 7 维可执行指令 |
| 英文靠二次翻译 | 中英双版同时输出 |
| 质量忽高忽低 | 同骨架 = 同基线 |

> Prompt 的乐高系统 —— 模具不变，材料自由替换。
